# YoutubeDownloader

Youtube üzerinden yt-dlp kullanılarak tekli veya çoklu(oynatma listesi)  video, mp3 indirilebilmesine olanak sağlıyor. Programın çalışabilmesi için gerekli yazılımların kurulması ve yt-dlp.exe dosyasınında bulunması gerekmektedir.
